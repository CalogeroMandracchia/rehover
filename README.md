# Rehover 👐

React hovering on two elements made simpler!

#### Install

```sh
yarn add rehover
```

#### Use it!

```js
import React from "react";
import Rehover from "rehover";
```

```jsx
    <Rehover delay={150}>
      <button source>
        Hover me!
      </button>
      <div destination>
        <a href="#">A</Item>
        <a href="#">B</Item>
        <a href="#">C</Item>
      </div>
    </Rehover>
```
